About

What is it?

Dinoteeth is a small footprint ripped media player that only depends on python and pyglet (plus some external media player like mplayer). Its goal is to run on the Raspberry Pi ARM linux machine.

Conceptually it’s like mythtv or xbmc, but without the ability to record anything. It’s designed for ripped media only.

Its strength is the ability to handle bonus features of ripped DVDs, which I found to be lacking in other media managers.

I don’t intend to compete with mythtv or xbmc at all. If you find this useful, then great! If not, no big deal; I wrote it as an experiment. :)

About the name

The name “dinoteeth” is an homage to Dinosaur’s Binket, a board book by Sandra Boynton that we’ve enjoyed reading to our son.
